# undotree

Visual undo history tree viewer.

## Usage

| Key         | Description     |
| ----------- | --------------- |
| `<leader>u` | Toggle undotree |

## Features

- Visual representation of undo history
- Navigate and restore previous states
- View diff between undo states
- Persistent undo across sessions (when undofile is enabled)

## Configuration

Located in: `lua/plugins/undotree.lua`

Lazy-loaded on the `UndotreeToggle` command.
