# which-key.nvim

Displays a popup showing available keybindings when you press a leader key.

## Usage

Press `<leader>` and wait briefly - a popup will show all available keybindings.

## Key Groups

| Prefix      | Group          |
| ----------- | -------------- |
| `<leader>a` | AI             |
| `<leader>c` | Code actions   |
| `<leader>d` | Debug          |
| `<leader>f` | Find/search    |
| `<leader>g` | Git operations |
| `<leader>h` | Git hunks      |
| `<leader>j` | Exit/quit      |
| `<leader>l` | Lua operations |
| `<leader>o` | Obsidian notes |
| `<leader>r` | Harpoon        |
| `<leader>x` | Diagnostics    |
| `<leader>y` | Terminal       |
| `<leader>z` | Toggles        |
| `gr`        | LSP            |
| `gs`        | Surround       |

## Features

- Zero delay popup
- Icon support (Nerd Font)
- Visual mode key groups

## Configuration

Located in: `lua/plugins/which-key.lua`
